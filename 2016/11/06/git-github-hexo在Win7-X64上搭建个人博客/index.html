Git+Github+Hexo在Win7_X64上搭建个人博客

时光易逝，记忆也不会永恒。

1.1 序
　　在搭建这个个人博客之前，我所有的笔记都是以word、txt或者其它形式的本地文档保存的。这样存在两个主要的问题：  
　　①各种文档的格式排版并不统一，即使是同一个软件如word文档，不同版本（03、07、10）兼容性也有问题，打开后会出现混乱的格式。  
　　②本地文件容易丢失，并且不能异地编辑。  
　　因此参考一些网上教程后，完成了自己博客的搭建，这下又可以去装逼了。扯远了，跟我一起去折腾吧。
